# 🍽️ Bites - Food Menu Landing Page

A modern and responsive Food Menu Landing Page built using HTML, Bootstrap 5, and CSS. The website showcases delicious food items with an attractive UI and mobile-friendly design.

## 🚀 Features

- Responsive Navigation Bar
- Food Category Filter Buttons
- Responsive Food Cards
- Star Rating
- Price and Add to Cart Button
- Newsletter Subscription Section
- Social Media Icons
- Responsive Footer
- Mobile Friendly Design

## 🛠️ Technologies Used

- HTML5
- Bootstrap 5
- CSS3
- Bootstrap Icons / SVG Icons
